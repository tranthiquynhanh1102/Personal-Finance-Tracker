# Finora - Personal Finance Tracker

A clean, modern, and minimalist personal finance web application built with vanilla HTML, CSS, and JavaScript.

![Finora Preview](https://via.placeholder.com/800x400/6366f1/ffffff?text=Finora+Finance+Tracker)

## ✨ Features

- **Real-time Balance Tracking** with elegant summary cards
- **Add Income & Expenses** through a simple, intuitive form
- **Transaction History** with date, description, and amount
- **Delete individual transactions** or clear all data
- **Data Persistence** using `localStorage` (data remains after refresh)
- **Fully Responsive** design with premium glassmorphism aesthetic
- **Beautiful UI** with clean typography and smooth interactions

## 🛠️ Tech Stack

- **HTML5**
- **CSS3** (Flexbox, Grid, Glassmorphism, modern gradients)
- **Vanilla JavaScript** (ES6+)
- **Font Awesome** icons
- **localStorage** for data persistence

## ⏱️ Project Duration
**Total time:** Approximately **5 hours**

## 🚀 How to Run

1. Clone or download this repository
2. Open the project folder
3. Double-click `index.html` or open it with Live Server
4. Start adding your transactions!

No installation or build process required.
